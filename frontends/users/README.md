# App Template

A clean React + MUI + Zustand starter template.

## Stack

- **React 18** with TypeScript
- **Material UI 5** for components and theming
- **Zustand** for state management
- **Vite** for bundling and dev server

## Getting Started

```bash
pnpm install
pnpm dev
```

Dev server runs on [http://localhost:3004](http://localhost:3004).
